Privacy Policy — DrawSession

Last updated: February 25, 2026

Hosted on GitHub Pages: https://melquisedeque.github.io/drawsession/

Your privacy is important to us. This Privacy Policy explains how DrawSession collects, uses, and protects personal information when you use our web application hosted on GitHub Pages.

1. Information We Collect

DrawSession only collects the data necessary to provide the service:

  • Pinterest authentication data: Access token from Pinterest API to access your boards and pins.
  • Session configuration data: Stored locally in your browser (Local Storage). No server storage.

We do not collect passwords, emails, or personal data.

2. How We Use Your Information

  • Authenticate via Pinterest OAuth
  • Display your selected images
  • Manage drawing session timers

All processing happens client-side. No data sent to servers.

3. Data Storage and Security

  • No backend servers: Data stored only in browser Local Storage
  • GitHub Pages: Static hosting only — no access to your data
  • HTTPS encryption for all Pinterest API calls

4. Third-Party Services

Only Pinterest API integration. Complies with Pinterest Terms of Service.

5. Data Deletion

Clear browser Local Storage or log out to delete all data instantly.
